# Data Folder

This folder contains four CSV files used for the Power BI dashboards in the grocery sales forecasting project.

The files were generated from two different machine learning pipelines:

- Training Pipeline
- Prediction Pipeline

These datasets are imported into Power BI for visualization and analysis.

---

# Prediction Pipeline Files

The following files were generated from the prediction pipeline notebook:

- `power_data_store_family_test.csv`
- `power_data_store_test.csv`

## Description

These files contain predicted sales values generated by the XGBoost forecasting models.

The models were trained using historical sales data from:

15/08/2016 to 15/08/2017

The trained models were then used to forecast sales for the period:

16/08/2017 to 31/08/2017

## Purpose in Power BI

These files are used to:

- Visualize forecasted sales trends
- Analyze predicted sales by store
- Analyze predicted sales by product family
- Support future sales forecasting dashboards

---

# Training Pipeline Files

The following files were generated from the training pipeline notebook:

- `power_data_store_family.csv`
- `power_data_store.csv`

## Description

These files contain both actual sales values and predicted sales values generated during model evaluation.

The datasets are used to compare the model’s predictions against actual sales data to evaluate forecasting performance.

The comparison period is:

23/05/2017 to 15/08/2017

## Purpose in Power BI

These files are used to:

- Compare actual sales and predicted sales
- Evaluate forecasting accuracy
- Visualize model performance over time
- Analyze prediction performance by store and product family

---

# Additional Note

The prediction pipeline notebook and the complete forecasting workflow are explained in detail in the main `README.md` file located in the project repository.
